# Production Analytics Project

This is a beginner SQL project to practice basic data analysis.

## Database
- PostgreSQL

## Files
- `01_create_tables.sql` – creates the production table
- `02_insert_data.sql` – inserts sample production data
- `03_analysis.sql` – contains simple queries:
  - show all data
  - difference between planned and actual
  - filter by product
  - filter by planned quantity range

## How to run
1. Run `01_create_tables.sql`
2. Run `02_insert_data.sql`
3. Run `03_analysis.sql` to see the results
